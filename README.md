# ऋचाएं — Richa Sharma Stories & Blogs

A beautiful Hindi & English story/blog platform by **Richa Sharma**.

## 📁 Project Structure

```
richa-website/
├── index.html        → Main homepage (Stories, Comics, Drama, Series)
├── story.html        → Individual story reading page
├── login.html        → Admin login page
├── editor.html       → Admin story editor
├── css/
│   └── index.css     → All styles (Neon theme + Category fonts)
└── js/
    ├── app.js        → Story rendering logic
    ├── auth.js       → Authentication (Login/Signup)
    └── db.js         → Database (InsForge backend)
```

## ✨ Features

- 🌙 Dark Neon Theme
- 🗣️ Hindi + English Support
- 🎭 Category-based Font Styles (Comics, Story, Drama, Series, Articles)
- 💬 Comments System
- 🔐 Admin Panel

## 🚀 Deployed on

- **Frontend:** Render / Netlify
- **Backend:** InsForge (Database + Auth)

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- InsForge (Backend as a Service)
- Google Fonts (Baloo 2, Hind, Bangers, Lora, Merriweather, Rajdhani)
